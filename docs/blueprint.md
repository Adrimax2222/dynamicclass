# **App Name**: ClassConnect

## Core Features:

- User Registration: Form for new users to input name, email, educational center, age range, role (teacher/student), profile picture (upload or predetermined), and admin code for class group joining.
- Data Storage: Store user registration data to host database.
- Loading Screen: Display loading screen while content loads, data is sent, and user registration completes.
- Home Screen: Main title 'Dynamic Class' with version number, dark/light theme switch, welcome message, trophy display for each student, and panels indicating number of homework, exams, tasks, and upcoming classes with details like grades and teachers. Navigation panel always visible.
- Personal Calendar: Calendar that syncs with Google Workspace Calendar API. Toggle between personal and class calendar views. Option to add new tasks and events with descriptions. Task descriptions available on click.
- AI Chatbot: Intelligent chatbot using the Gemini API, personalized for education. Options to adjust response length and subject/topic. Users can upload files and audios. Toggle bubble to access the chatbot from any screen.
- Personalized recommendations with reasoning: Generative AI tool recommends courses that match student's ongoing classes.

## Style Guidelines:

- Primary color: Deep blue (#2962FF), evoking trust and intelligence.
- Background color: Very light blue (#E0F7FA).
- Accent color: Violet (#7C4DFF) to create contrast and emphasis.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, and readable feel.
- Use educational and class-related icons for clarity and engagement.
- Clean and structured layout to display class schedules, tasks, and chatbot clearly.
- Subtle transitions and animations on screen changes and when tasks are added or completed.
